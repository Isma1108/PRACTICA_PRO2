\doxysubsection{Descripción detallada}
Struct que almacena información sobre una categoria. 

Como atributos no se incluye el identificador entero entre 1 y C porque éste será el índice del vector que almacene el conjunto de categorias. El vector de enteros de puntuacion\+\_\+por\+\_\+nivel contiene la información de los puntos que ganaria un jugador en un torneo de categoria c al llegar a un nivel k, siendo k un índice de este vector.
